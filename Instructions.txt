1. This code uses Python OpenCV alongside Python SpeechRecognition 3.1.3, to achieve face detection by speech input.

2. To install SpeechRecognition 3.1.3 on your linux system, type the following commands:

3. sudo apt-get install python-pyaudio python3-pyaudio

4. sudo apt-get install python-pip

5. sudo pip install SpeechRecognition
